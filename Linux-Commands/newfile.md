### Cat Commands
```bash
To type into the terminal
cat > demofile.txt
type
and CTRL + C to quit and save

cat << EOF > anything.txt
typing anything until EOF
Type EOF to end the and save the file

cat << STOP > demo.txt
This can be anything in the between borders
To stop just type STOP

```
